# Lookup and Print Cheat Sheet

## Start the Station
1. Open **Settings** on this browser.
2. Under **This Device**, choose **Lookup & Print**.
3. Enter the admin PIN to enable kiosk mode.
4. Confirm the header now says **Lookup & Print Station**.

## Finding the Right Order
- Use the station search box for **customer name**, **order number**, or **pickup code**.
- The page is intentionally limited to safe volunteer actions.
- If nothing matches, stay on the station page and try the search again. Do not create a walk-up order from this station.

## Printing
1. Click **Print Order Sheet** to open the order paperwork in a new tab.
2. Click **Print Seller Packet** only when that button is shown for the order.
3. Close the print tab when finished. The original station tab stays open.

## Tips
- Allow pop-ups if print preview does not open.
- This station does not edit orders, run imports, open reports, or open settings.
- Use **Admin Unlock** in the kiosk header if a lead needs the full app.
