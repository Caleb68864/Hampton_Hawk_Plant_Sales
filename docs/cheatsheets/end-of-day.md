# End of Day Cheat Sheet

## Step 1: Check Remaining Orders
1. Go to **Orders** and filter by status **Open** or **InProgress**
2. Attempt to fulfill any remaining orders at the pickup station
3. For no-shows or unavailable items, use **Force Complete** with admin PIN

## Step 2: Review Problem Orders
1. Go to **Dashboard** and check the problem orders section
2. Resolve any flagged issues (over-fulfilled, stalled orders)
3. Use **Reset** if an order needs to be re-scanned

## Step 3: Check Inventory
1. Go to **Inventory** to review remaining stock
2. Note any items with zero or negative quantities for reconciliation
3. Use the **Low Inventory** report on the Dashboard for a quick view

## Step 4: Close the Sale
1. Go to **Settings**
2. Click **Close Sale**
3. Enter the admin PIN
4. Provide a reason (e.g., "End of sale day")
5. This prevents any further order creation or modification

## Step 5: Print Final Reports
- Print the **Dashboard** metrics for records
- Print any remaining **Order Sheets** for pickup
- Print **Seller Packets** if needed for seller credits

## Step 6: Verify
- Confirm all orders are in **Complete** or **Cancelled** status
- Check that the sale is showing as **Closed** in Settings
- The system will display a "Sale Closed" banner on all pages

## After Closing
- Fulfillment scanning still works for late pickups
- Admins can reopen the sale if needed (requires PIN)
- All data is preserved -- nothing is deleted
