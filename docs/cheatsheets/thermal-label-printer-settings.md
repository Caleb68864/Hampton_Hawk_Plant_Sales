# Thermal Printer Settings for Plant Labels (1" x 2")

Use these settings when printing plant barcode labels from **Print Plant Labels**.

## Recommended Defaults

- **Label size:** 2.00 in wide × 1.00 in high
- **Media type:** Gap / die-cut labels
- **Orientation:** Landscape (if your printer defines width as feed direction)
- **Scale:** 100% (disable "Fit to page" and "Shrink to printable area")
- **Margins:** 0 or minimum supported by driver
- **Color mode:** Black & White / Monochrome
- **Print quality / darkness:** Start at medium, increase only if bars look gray
- **Speed:** Medium or slower for sharper bar edges

## Browser Print Dialog Checklist

1. Open plant labels from Plants list or plant detail page.
2. Click **Print Labels**.
3. Select your thermal printer.
4. Open **More settings** and verify:
   - Background graphics: **On**
   - Margins: **None** or **Minimum**
   - Scale: **100**
5. Print one test label before large runs.

## Calibration Notes

If labels drift up/down or print across gaps:

1. Run the printer's **Media Calibration** or **Gap Sensor Calibration** from the printer utility.
2. Confirm the exact stock size in the printer driver matches 2"×1".
3. Clean the printhead with approved thermal cleaning wipes.
4. Re-run a test print and verify barcode bars are crisp and centered.

## Barcode Scannability Checks

- The full barcode and text must be visible (no clipping).
- Bars should be solid black (no fading).
- Quiet zone (blank space) should remain on both sides of barcode.
- Test with the same scanner model used at pickup before final print runs.
